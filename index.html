Data Structure

Introduction

The course is one of core courses for the major of computer science and technology. It's foundation to design and realize the compiler, operating system, database system and other system software and application software.

This course attaches importance to learning fundamental knowledge and training data structures and algorithms application ability. The core course contents is data description and data processing, include how to organize data reasonably, and how to store and process data efficiently, and how to design, analyze and evaluate algorithms correctly.

This course aims to make students more comprehensive grasp of all kinds of commonly used data structure, provide the necessary foundation for learning subsequent software course, grasp and constantly improve the ability to use data structure to solve practical problems.
